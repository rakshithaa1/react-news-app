# 📰 React News App

A modern, responsive news application built with React that allows users to browse the latest news, search articles, bookmark favorites, view recently viewed articles, and switch between light and dark mode.

---

## 🚀 Live Demo

https://react-news-app-two-rose.vercel.app

---

## ✨ Features

- Browse latest news
- Search articles
- Filter news by category
- Bookmark articles
- Remove bookmarks
- Recently viewed history
- Dark Mode
- Responsive design
- Loading spinner
- Error handling
- Image fallback
- Persistent data using Local Storage

---

## 🛠️ Tech Stack

- React
- React Router DOM
- Vite
- CSS
- React Icons
- GNews API

---

## 💾 Installation

```bash
git clone https://github.com/rakshithaa1/react-news-app.git

cd react-news-app

npm install

npm run dev
```
